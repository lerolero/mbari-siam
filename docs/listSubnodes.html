NAME
listSubnodes - list subnodes of specified MOOS node
SYNOPSIS 
listSubnodes nodeHostname
DESCRIPTION
The listSubnodes utility lists information about subnodes of the specified MOOS node, including the subnode IP address, its ISI ID, and "age", which is time elapsed since the subnode last notified the parent node of its existence.  The address of a network switch attached to the subnode is also displayed, if the subnode is currently running a NetworkSwitch service (e.g. the Medusa service).

Example output listSubnodes is shown below:.  
% listSubnodes loc
SIAM version $Name: HEAD $
Found 4 Subnodes
node402/134.89.36.2: deviceID=1598, age=3m:33s
Switch address: 134.89.36.53/134.89.36.53

node413/134.89.36.13: deviceID=1410, age=1m:31s
Switch address: 134.89.36.63/134.89.36.63

node418/134.89.36.18: deviceID=1522, age=1m:6s
Switch address: 134.89.37.65/134.89.37.65

node403/134.89.36.3: deviceID=1544, age=5m:49s
Switch address: 134.89.36.52/134.89.36.52

ENVIRONMENT

The following environment variables must be set:
  • JAVA_HOME - location of Java installation, e.g. /j9 or /opt/java/jdk1.3.1
  • JAVA - name of the JVM. E.g. on a Sidearm, JAVA should be set to j9. On a workstation with Sun's Java installed, JAVA could be set to java.
  • SIAM_HOME - top of the SIAM directory tree
  • SIAM_CLASSPATH - on a Sidearm, SIAM_CLASSPATH should be set to $JAVA_HOME/lib/jclMax/classes.zip:$JAVA_HOME/lib/prsnlmot.jar:$JAVA_HOME/lib/RXTXcomm.jar:$SIAM_HOME/classes.  On X86 system, SIAM_CLASSPATH should be set to $SIAM_HOME/classes
