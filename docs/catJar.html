NAME
catJar - list the contents of a jar file
SYNOPSIS 
catJar jarFile [x]
x - the 'x' option displays contents of service.xml if present
DESCRIPTION
The catJar utility lists the contents of a jar file including the entries in the jar file, the entries in the service.properties file if it exists, and the manifest entries. To use catJar enter the command followed by the name of the jar file, for example 
$ catJar puck.jar
This utility may be run on the MMC as well as a Windows or Linux PC. 





ENVIRONMENT

The following environment variables must be set:



    

  • 
JAVA_HOME - location of Java installation, e.g. /j9 or /opt/java/jdk1.3.1
    • 


  • 
JAVA - name of the JVM. E.g. on a Sidearm, JAVA should be set to j9.
On
a workstation with Sun's Java installed, JAVA could be set to java.
    • 


  • 
SIAM_HOME - top of the SIAM directory tree
    • 


  • 
SIAM_CLASSPATH - on a Sidearm, SIAM_CLASSPATH should be set to SIAM_CLASSPATH=$JAVA_HOME/lib/jclMax/classes.zip:$JAVA_HOME/lib/prsnlmot.jar:$JAVA_HOME/lib/RXTXcomm.jar:$SIAM_HOME/classes:$SIAM_HOME/jars/log4j-1.2.8.jar. 
On X86 system, SIAM_CLASSPATH should be set to $SIAM_HOME/classes:$SIAM_HOME/jars/log4j-1.2.8.jar
