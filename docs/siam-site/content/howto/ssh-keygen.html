Name

SSH Keygen HOWTO

Synopsis

Configure security keys for password-less SSH. This enables trusted processes to transfer files from node to node or between portal and node.

Configure portal - node

Set up 'no password' login from user1@portal to user2@node

cd ssh-keygen -t dsa chmod 0644 ~/.ssh/id_dsa.pub scp ~/.ssh/id_dsa.pub user2@node:.ssh/id_dsa.pub

Then log onto node as user2:

ssh user2@node cd ~/.ssh cat id_dsa.pub >> authorized_keys2

Configure node - node

Set up 'no password' login from user1@ node1 to user2@node2

On node1, as user1:

cd ssh-keygen -d chmod 0644 ~/.ssh/id_dsa.pub scp ~/.ssh/id_dsa.pub user2@node2:.ssh/id_dsa.pub

Then log onto node2 as user2:

ssh user2@node2 cd .ssh cat id_dsa.pub >> authorized_keys2
