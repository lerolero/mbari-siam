Name

BIN Power HOWTO

Synopsis

Procedures for switching power to MOOS BINs

Shutdown SHELF BIN

Power down instruments with internal batteries

  • Minicomm to RDI Sentinal ADCP and send break character to halt sampling. Send command: “CZ” to sleep instrument.
  • Wetlabs – OK.
  • SBE 16?
  • Nortek

Halt SHELF BIN (disable backup ultra-capacitors and batteries, exit SIAM, sync filesystem, halt processor)

killBIN node1 -q

Shut down 400 volts to SHELF BIN from SURFACE node using

switchBIN400V surface 0 OFF

Power SHELF BIN

Switch SURFACE 400V power on

switchBIN400V surface 0 ON

Power AXIS BIN

Power up SHELF BIN as above

Make sure backup ultra-caps and batteries are enabled on SHELF BIN

binBackups node1 /dev/ttySX13 ALLEN

Power down SHELF BIN 400V (controlled by SURFACE node) for not over 2 minutes. If power is off more then two minutes node will be depleting back-up batteries.

switchBIN400V surface 0 OFF

Immediately switch 400V AXIS BIN power on (controlled by SHELF BIN)

switchBIN400V node1 /dev/ttySX13 0ON

Immediately switch 400V power to SHELF BIN on (controlled by surface)

switchBIN400V surface 0 ON

Shutdown AXIS BIN

Shutdown instrument services for the following instruments.

commands

Enable power and communications to the instruments

powerUpPort <node> <port>
node: node hostname, e.g., node1, node5, etc.
port: port number, e.g., 7, /dev/ttySX4, etc.

Stop sampling in instruments with internal batteries

  • Minicom to RDI Sentinel ADCP and send break character to halt acquisition/logging. Send command: “CZ” to sleep instrument.
  • WETLabs ECO Triplet. Automtically goes to low-power state.
  • SBE-16+ and SBE-37SM - Automatically goes to low-power sleep after 2 minutes without receiving serial commands from host. To be sure, can a issue "STOP" command to instruments through 'stip' command
  • Nortek Aquadopp. Attach test cable to PC, use GUI to ensure that data streaming/logging was turned off.

Halt AXIS BIN (disable backup ultra-capacitors and batteries, exit SIAM, sync filesystem, halt processor)

killBIN node5 -q

Switch 400 volts to AXIS BIN off (controlled by SHELF BIN)

switchBIN400V node1 /dev/ttySX13 0OFF

Shutdown Snubber Node

Sync filesystem and halt processor

killnode node10 -q

Shutdown power to SNUBBER (controlled by instrument service on SURFACE port /dev/ttySX1)

shutdownPort surface 1
