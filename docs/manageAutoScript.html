NAME
manageAutoScript - automatically transfer autoscript from portal to MOOS surface node; retrieve autoscript output log from surface node.
SYNOPSIS 
manageAutoScript localSiamHome surfaceSiamHome mailRecipient(s)
DESCRIPTION
manageAutoScript retrieves available autoscript output from surface node to shore, and then automatically transfers a "pending" autoscript from shore to surface node. manageAutoscript first retrieves the latest log file generated by runAutoScript from the surface node, and copies it to localSiamHome/autoScript/logs/latest..  manageAutoscript then looks for a local "pending" file named localSiamHome/autoSript/pending/autoscript, and transfers it via scp to surfaceSiamHome/autoScript/pending/autoscript. Once transferred successfully,  the autoscript contents are appended to file localSiamHome/autoScript/copied/copied.log. manageAutoScript then sends a status message to recipients specified in the third command-line argument.

Note that the last line of an autoscript must be the following special comment:

# AUTOSCRIPT EOF

The '#' commant character must appear in the first column.

NOTE: ssh MUST be configured such that user 'ops' is not prompted for a password when transferring files via scp to the surface node.

To automatically execute manageAutoScript when a ppp link is established between shore and MOOS surface node, you must add an appropriate entry in /etc/ppp/ip-up.local. For example:
ENVIRONMENT
This script does not depend on any environment variables.
